# portugol-docs

Repositório contendo o código do site de documentação da minha versão de Portugol.

🌐 **Acesse a documentação em:** https://gabriel-v-maia.github.io/portugol-docs/

## Rodando localmente

```bash
pip install mkdocs-material
mkdocs serve
```

Acesse no link que aparecer no terminal.

## Contribuindo

Foque o projeto e crie uma branch pro seu pull request. Inclua uma descrição do que foi adicionado e, se necessário, um texto explicando o motivo da mudança.

Tipos de contribuição bem-vindos:

* Corrigir exemplos incorretos
* Documentar features que ainda não têm página
* Melhorar explicações existentes

## Interpretador

Este site documenta o [Interpretador Portugol](https://github.com/Gabriel-V-Maia/Interpretador-Portugol), o projeto principal da linguagem.
