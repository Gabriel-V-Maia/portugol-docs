# Expressões

## Operadores aritméticos

| Operador | Operação        |
|----------|-----------------|
| `+`      | Adição          |
| `-`      | Subtração       |
| `*`      | Multiplicação   |
| `/`      | Divisão         |
| `%`      | Módulo          |

## Operadores relacionais

| Operador | Significado     |
|----------|-----------------|
| `==`     | Igual           |
| `!=`     | Diferente       |
| `<`      | Menor           |
| `>`      | Maior           |
| `<=`     | Menor ou igual  |
| `>=`     | Maior ou igual  |

## Operadores lógicos

| Operador | Significado |
|----------|-------------|
| `e`      | AND         |
| `ou`     | OR          |
| `nao`    | NOT         |

## Interpolação de strings

Qualquer expressão pode ser embutida em uma string com `${}`:

```portugol
escreva("Soma: ${a + b}\n")
```
