# Tipos de Dados

| Tipo      | Descrição                  | Exemplo          |
|-----------|----------------------------|------------------|
| `inteiro` | Número inteiro             | `inteiro x = 10` |
| `real`    | Número de ponto flutuante  | `real pi = 3.14` |
| `cadeia`  | Texto (string)             | `cadeia s = "oi"`|
| `logico`  | Verdadeiro ou falso        | `logico b = verdadeiro` |
| `nulo`    | Ausência de valor (retorno)| `nulo funcao f()`|

## Declaração de variáveis

```portugol
inteiro idade = 20
real altura = 1.75
cadeia nome = "Gabriel"
logico ativo = verdadeiro
```
